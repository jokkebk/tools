# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is a collection of standalone single-file HTML web tools and games, hosted at tools.codeandlife.com via GitHub Pages. Each tool is a self-contained HTML file with embedded CSS and JavaScript - no build system or npm dependencies.

## Architecture

- **Tools**: Each `.html` file is a complete, standalone web application
- **Documentation**: Sidecar `.md` files contain tool descriptions (first paragraph is used as summary in README)
- **Auto-generated files**: `README.md` and `tools.json` are generated by CI - do not edit manually

## Adding New Tools

1. Create a new `.html` file with all CSS/JS embedded
2. Create a matching `.md` file with a description paragraph
3. Commit and push - GitHub Actions automatically regenerates the index

## Build System

The README.md is auto-generated on push to main:

```bash
python3 gather_tools.py    # Scans HTML files and creates tools.json with git metadata
python3 build_index.py     # Generates README.md from tools.json and .md sidecar files
```

Run locally to preview changes before push.

## Git Workflow

Use `./push.sh` for pushing to handle potential rebase conflicts from auto-generated files:
```bash
./push.sh   # Runs: git pull --rebase origin main && git push origin main
```
